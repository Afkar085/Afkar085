## Projects

**Personalized Meal Plan Recommendation System**
`Python · FastAPI · Scikit-Learn · Pandas · K-Means`

Clustered 500+ food items across 6 nutrition features using K-Means into 5 BMI-based groups. Built a FastAPI endpoint that generates a full 30-day meal plan on demand. Tested with 20 users — 85% said the plans felt realistic and usable.

---

**Smart Shelf Inventory System** — [repo](https://github.com/Afkar085/Smart-Shelf)
`PostgreSQL · PL/pgSQL · Java · JDBC · HTML · CSS · JavaScript · Supabase`

Inventory management system with a normalized DB schema, PL/pgSQL triggers for automatic expiry alerts and invalid-entry prevention, and a Java-JDBC backend for real-time transactions. Web dashboard for live item tracking with status indicators.

---

**Robot Path Planning with IDA\***
`Python · ROS 2 · Nav2 · OpenCV · RViz2`

Implemented IDA* on a 100×100 grid — gets optimal paths with significantly lower memory than A*. Integrated into ROS 2 Nav2 and tested on TurtleBot3 and Unitree Go2. Visualized traversal in RViz2.
